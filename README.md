# Carbon Emission Tracker (DBMS Project)

A professional Full-Stack web application built to track and visualize daily carbon footprints from transportation.

## 🚀 Features
- **Full CRUD:** Create, Read, Update, and Delete transport logs.
- **Authentication:** Secure Login and Logout system.
- **Data Visualization:** Dynamic Bar/Pie charts using Chart.js.
- **Responsive UI:** Built with Bootstrap for a clean, professional look.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Visuals:** Chart.js

## 📋 Installation & Setup
1. Download and install [XAMPP](https://www.apachefriends.org/index.html).
2. Clone this repository into `C:\xampp\htdocs\`.
3. Open **phpMyAdmin** and create a database named `carbon_tracker`.
4. Import the provided `.sql` file to create the tables.
5. Start Apache and MySQL in the XAMPP Control Panel.
6. Open your browser and go to `localhost/carbon_tracker/index.php`.
